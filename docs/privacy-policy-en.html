Privacy Policy

Last updated: April 14, 2026

Overview

"URL Query Editor" (hereinafter referred to as "the Extension") respects and protects user privacy. This Privacy Policy explains how the Extension handles your data.

Data Collection

The Extension does not collect, store, or transmit any personal data or user information.

Data Usage

The Extension only reads the URL of the current tab when you actively use it, so that you can edit its query parameters. All operations are performed locally in your browser and no data is sent to any external server.

Third-Party Services

The Extension does not use any third-party analytics, advertising services, or tracking technologies.

Permissions

  • activeTab: Used to read and modify the URL query parameters of the current tab.
  • tabs: Used to update the URL of the current tab.

Policy Changes

Any changes to this Privacy Policy will be reflected on this page.

Contact

For any privacy-related questions, please reach out via GitHub Issues.
